HashiCorp Certified: Terraform Associate

Resumen del Proyecto

Certificación oficial emitida por HashiCorp que valida una experiencia integral en Infraestructura como Código (IaC). El alcance cubre la gestión de recursos en nubes públicas, el flujo de trabajo estándar de Terraform (init, plan, apply, destroy), gestión segura del estado (state), modularización y el uso avanzado de variables y funciones para diseñar entornos multi-cloud resilientes.

Herramienta de CLI GoGoVersion

Resumen del Proyecto

Una herramienta nativa de interfaz de línea de comandos (CLI) desarrollada en Go para optimizar los flujos de automatización de lanzamientos (releases). Diseñada específicamente para integrarse de forma fluida en pipelines de CI/CD, permitiendo calcular, incrementar y etiquetar versiones de software automáticamente basándose en Versionado Semántico (SemVer) o convenciones personalizadas de Git con un rendimiento ultrarrápido.

Arquitectura / Core

Estructura Base de Microservicios (Core)

Resumen del Proyecto

La columna vertebral estructural y ecosistema central diseñado para estandarizar y acelerar el desarrollo de microservicios empresariales. Impone principios de Arquitectura Limpia (Clean Architecture) y Arquitectura Hexagonal, aislando estrictamente las capas de dominio, aplicación e infraestructura. Incluye configuraciones de Docker listas para producción y envoltorios (wrappers) de abstracción compartidos para servicios backend.

Arquitectura / REST

Plantilla de Microservicios: API REST

Resumen del Proyecto

Plantilla base de nivel empresarial especializada en la construcción de microservicios que exponen interfaces HTTP/REST. Desarrollada siguiendo los flujos de Arquitectura Limpia, implementa de forma nativa enrutamiento optimizado, middlewares transversales esenciales (registro estructurado, políticas CORS, limitación de tasa y métricas), validación estricta de payloads JSON y un formato de respuesta HTTP estandarizado.

Arquitectura / gRPC

Plantilla de Microservicios: Servicios gRPC

Resumen del Proyecto

Plantilla base de alto rendimiento diseñada específicamente para la comunicación sincrónica de baja latencia entre servicios (tráfico este-oeste) utilizando gRPC. Cuenta con una configuración automatizada del compilador de Protocol Buffers (Protobuf) para definir estrictamente los contratos de servicio (`.proto`) y optimiza la serialización eficiente de datos y el streaming sobre conexiones persistentes HTTP/2.

Cloud Native / DevOps

Guía de Estudio de Kubernetes CKA

Resumen del Proyecto

Un manual de ingeniería exhaustivo y registro práctico de laboratorio dedicado a la preparación del examen Certified Kubernetes Administrator (CKA) de la Cloud Native Computing Foundation (CNCF). Incluye scripts de despliegue automatizados, laboratorios de clusters locales, hojas de trucos avanzadas de `kubectl` e investigaciones profundamente documentadas sobre componentes del core, redes (CNI), almacenamiento (CSI), seguridad (RBAC), resolución de problemas y respaldos de etcd.
